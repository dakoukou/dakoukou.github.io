类，面向对象的三大属性

Mar 7, 2018 | java

类：对事物的描述

  • 对象：一切皆对象

  • 面向对象：三大属性：封装，继承，多态

    1.创建对象: new 找对象

  • 类名 对象名=new 类名（）；

  • 使用对象操作方法： 对象名称.方法（参数）

2.变量

  • 全局变量（成员变量）和局部变量

  • 全局变量：作用范围 在当前类中有效

  • 局部变量 只在定义他的方法中，或方法体有效

3.堆和栈

  • 堆包含了很多的栈

  • 实例化对象–>创建了一个堆内存，对象运行在堆内存中，全局变量也运行咋i这个堆内存中

  • 局部变量–>只运行在栈内存中
