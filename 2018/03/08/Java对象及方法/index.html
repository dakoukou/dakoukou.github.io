对象及方法

  • 对象：一切皆对象，对象是由属性的

  • 方法：对某些属性或运算的一个封装

  • 方法的作用：逻辑运算或对运算结果的传递

    变量的限制属性：

    1.public：公开的属性
    2.private：私有的属性 只是在当前类中调用
    3.protected ：具有保护的属性 只可以在父类进行调用

    • default ：默认的属性 缺省属性
