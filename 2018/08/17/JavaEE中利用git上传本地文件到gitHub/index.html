利用git上传文件的具体步骤

我在这之前上网查了很多博主的资料，不过没找到很具体的，所以写个攻略给需要的人看。

操作步骤：

1.下载git工具

git下载链接 选择适合的版本

2.安装完成，打开Git Bash，输入自己的用户名和邮箱

3.设置SSH Key

首先查看你是否已经生成了密钥

如果出现下面的结果，说明已经有了密钥

如果没有，接下来就需要生成密钥

  • 密钥类型这里用 -t 选项指定为rsa （默认生成用于SSH-2的RSA密钥）
  • 密钥中用-C来指定所指定的注释，可以方便用户标识这个密钥（这里输入自己的邮箱或者其他都行）
  • 输入完毕后会要求你输入一个passphrase，空表示没有密语，接着会让输入2次口令(password)，空表示没有口令。3次回车即可完成当前步骤。
  • 此时[c盘>用户>自己的用户名>.ssh]目录下已经生成好了。

4.在你的GitHub账号上配置你的SSH key

打开GitHub网站，登陆，点击右上角用户头像下的小三角，找到setting，在右侧菜单栏中找到SSH and GPG keys，选择new SSH key

点击 ADD SSH key，之后你的SSH key就配置成功了

5.选择一个本地文件夹作为你的库

(1) 在这个文件夹下右键点击 git Bash here

(2) 初始化

(3) 执行指令将文件添加到本地仓库：

6.在你的GitHub中新建一个仓库

复制仓库地址：git@github.com:dakoukou/liuzphi.git

7.Java EE 关联 GitHub

打开Java EE -> Window -> preferences ->General ->Network Connections ->ssh2

你会发现你的密钥已经在里面了

(1) 右键你要上传的项目 -> team ->share project

(2) 然后再次右键项目->team->commit

(3) 输入你的密钥的密码（没有不用），URL是你刚才复制的仓库地址 ，运行完你会发现你的程序已经出现在你的GitHub仓库中了

8 结束
